# mario-kart-gl

Recreation of a Mario Kart 64 item box with C and OpenGL.
Runs on macOS, Linux and Windows (as long as you have GCC and Make) thanks to [RGFW](https://github.com/ColleagueRiley/RGFW).

## Dependencies

- GCC + Make
- OpenGL 3.3

## Quick start

```console
$ make
$ ./mario-kart-gl
```
